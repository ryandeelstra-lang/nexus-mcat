# charged_up — Knowledge Garden, an MCAT study game forked from Anki

> **This is a fork of [Anki](https://github.com/ankitects/anki)** (AGPL-3.0-or-later), re-targeted as
> **Knowledge Garden** — a cozy pixel-art MCAT study game for **desktop + iOS that share one Rust engine**:
> answering flashcards plants, waters, and blooms a living garden, with three honest scores
> (Memory / Performance / Readiness) with ranges + a give-up rule, sourced & evaluated AI card generation,
> and two-way sync. See [NOTICE.md](./NOTICE.md) for attribution and
> [LICENSE](./LICENSE) for the (unchanged) AGPL-3.0-or-later terms. Forked from upstream `b00308e` (v26.05).
> Build/run instructions for both apps and the architecture overview live in the project planning docs.
